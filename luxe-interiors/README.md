# Luxe Interiors — demo website

Premium interior design studio marketing site: **Vite**, **React 19**, **TypeScript**, **Tailwind CSS v4**, **Framer Motion**, and **React Router**.

## Scripts

```bash
npm install
npm run dev      # local dev server
npm run build    # production bundle
npm run preview  # preview production build
```

## Customize for your client

- **Branding:** `index.html` (title/meta), `src/components/layout/Header.tsx`, `src/components/layout/Footer.tsx`
- **Copy & imagery:** `src/data/site.ts` and page files under `src/pages/`
- **Contact form:** `src/pages/Contact.tsx` — connect to your API, Formspree, Basin, or email service (currently a client-side demo success state)

## Notes

- Remote images use [Unsplash](https://unsplash.com/) URLs; replace with licensed project photography before launch.
- Social links in the footer are placeholders (`#`).
